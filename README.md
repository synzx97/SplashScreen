# SplashScreen Boot Animation KDE Plasma with live video
This is complate guide on how to configure my custom KDE Plasma SplashScreen. You can follow the step-by-step instructions provided to easily customize and enhance the appearance of yourSplashScreen.

This SplashScreen consists of two phases:
- Phase 1 – Displays the logo, text, and progress bar.
- Phase 2 – Plays a live video (or live wallpaper) right before entering the desktop.
